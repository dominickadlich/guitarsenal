# 🎸 Guitarsenal 🎸

A full-stack guitar collection manager for tracking instruments, string setups, and maintenance history — built with Django REST Framework and React.

## The Problem

Guitar collections grow untracked — string gauges, last-change dates, purchase history, and setup notes end up scattered across memory, sticky notes, or nothing at all.

## The Solution

Guitarsenal centralizes it all: instrument details, photo galleries, string setups, and a dated maintenance log, with search and filtering across the collection.

## Tech Stack

- **Backend:** Django, Django REST Framework
- **Frontend:** React, React Router
- **Styling:** Tailwind CSS
- **Database:** PostgreSQL (production) / SQLite (development)

## Status

Personal project — actively used to track my own collection.

## Why This Project Matters

Built end-to-end as a full-stack exercise outside a clinical context — a REST API with a relational data model, a React frontend consuming it, and file/image handling, all shipped independently.
